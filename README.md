# 个人记账管理系统 python1000

## 项目概述

个人记账管理系统是基于 Python Django 开发的，旨在帮助用户高效管理个人财务，包括记录和统计收入与支出。

## 技术栈

- **后端**: Python Django
- **数据库**: MySQL
- **前端**: HTML

## 功能模块

### 基础功能

- 用户登录
- 个人信息修改

### 财务管理

- 收入记录
- 支出记录
- 收入记账管理
- 支出记账管理

### 数据统计

- 收入统计
- 支出统计

## 系统角色

- 普通用户：负责日常的财务记录和查询

## 运行环境

- Python 环境配置
- MySQL 数据库服务

## 部署步骤

1. 安装 Python 和 Django
2. 配置 MySQL 数据库
3. 导入数据库文件
4. 运行 Django 项目
